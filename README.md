   ## ✨ Features

   - **Premium UI/UX**: Fully responsive, beautifully designed dark/light mode interface using modern web aesthetics and smooth `framer-motion` animations.
   - **Dynamic Content Engine**: Data is driven entirely by a PostgreSQL database, meaning no hardcoded strings. Add projects, update experiences, or swap roles instantly.
   - **Secure Admin Panel**: A private backend dashboard protected by JWT authentication with HTTP-only cookies.
   - **Cloud Media Management**: Integrated with Cloudinary for blazing-fast, optimized image and PDF delivery.
   - **Real-Time Analytics**: Built-in backend tracking for visitors, engagement rates, and incoming contact messages.
   - **Live Device Preview**: Instantly view how the portfolio responds to mobile, tablet, and desktop directly from the admin panel.
   - **SEO Optimized**: Built for high performance and global discoverability with dynamic meta tags.

   ---

   ## 🛠️ Architecture

   This portfolio operates as a full SaaS-style application:

   - **Frontend Application**: Built using the Next.js App Router for optimal Server-Side Rendering (SSR) and SEO.
   - **API Layer**: Secure REST endpoints manage authentication, content fetching, media uploads, and email notifications.
   - **Database**: PostgreSQL handles structured content (Projects, Settings, Messages) utilizing Prisma ORM for type-safe queries.

   ---

   ## 📂 Core Capabilities

   1. **Content Management System (CMS)**
      - Update Hero text, roles, and impact statements.
      - Add/Edit/Delete Projects, Work Experience, Education, and Certifications.
   2. **Media Library**
      - Upload global assets like the Site Favicon and Resume CV.
      - Automatically hosts and delivers assets via CDN.
   3. **Inbox & Communications**
      - Public visitors can submit inquiries via the Contact form.
      - Messages are stored securely in the database and emailed directly to the administrator.
